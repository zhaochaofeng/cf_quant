# Barra CNE6 因子体系

## 概述

Barra CNE6 是由全球知名指数与风险管理工具提供商 MSCI 于 2018 年发布的第六代中国股权市场模型（China Equity Model, Version 6）。

根据 Barra 模型，股票收益率可由国家因子、行业因子、风格因子三类因子上的暴露解释。CNE6 在 CNE5 基础上进行了细化和扩充，形成了由 **9类一级风格因子**、**20个二级基础因子**、**46个三级因子** 组成的风格因子体系。

CNE6 因子是经过长期验证稳定的因子，在收益率模型训练时通常必须加入。

### 作用

1. **量化个股系统性风险的组成**：实现主动选择所承受的风险
2. **绩效归因**：将个股收益按照多因子模型公式分解，alpha为正、统计显著且长期稳定则说明是能力，否则为运气
3. **拆解 CAPM 模型**：将单因子模型扩展为多因子模型

### 多因子模型公式

```
r_{i,t} = α_i + β_MKT · r_MKT,t + Σ(X_{i,j,t} · f_{j,t}) + Σ(X_{i,c,t} · f_{c,t}) + ε_{i,t}
```

其中：
- `r_i,t`：股票 i 在时间 t 的收益率
- `α_i`：阿尔法（截距项），真正的超额收益（能力）
- `β_MKT`：市场 BETA
- `r_MKT,t`：市场因子收益率
- `X_i,j,t`：股票 i 在风格因子 j 上的暴露度
- `f_j,t`：风格因子 j 在时间 t 的收益率
- `X_i,c,t`：股票 i 在行业 c 上的暴露度（0或1）
- `f_c,t`：行业因子 c 在时间 t 的收益率
- `ε_i,t`：特异收益（残差）

## 一、Size（规模）

### 1.1 LNCAP - 流通市值对数

**含义**

LNCAP（Log of Market Capitalization）用来解释规模与收益的关系，捕捉股票规模（市值）差异对收益率的影响。市场中通常存在"小盘股溢价"或"大盘股稳定性"特征。

**计算逻辑**

```
LNCAP = ln(股票流通市值)
```

**所用字段**

- 流通市值（万元）

---

### 1.2 MIDCAP - 中市值

**含义**

用于解释中型市值公司是否脱离大/小盘走出独立行情，捕捉市值与收益之间的非线性关系。

**计算逻辑**

1. 对原始的规模因子暴露（LNCAP）取立方，以捕捉市值与收益之间存在的非线性关系：
   ```
   x = (LNCAP)^3
   ```

2. 将上述非线性项 x 对原始的规模因子（LNCAP）进行线性回归，取残差。此步骤旨在剥离与线性规模效应相关的部分，确保 Mid Cap 因子与 Size 因子正交（不相关）：
   ```
   ε = x - α - β · LNCAP
   ```

3. 对正交化后的序列进行去极值处理（常用方法为中位数绝对偏差法），以控制极端值的影响：
   ```
   ε = Winsorize(ε)
   ```

4. 对去极值后的序列进行 Z-Score 标准化，使其均值为 0，标准差为 1：
   ```
   MIDCAP = (ε - μ_ε) / σ_ε
   ```

**所用字段**

- 流通市值（万元）

## 二、Volatility（波动率）

波动率因子核心作用是衡量股票价格波动的剧烈程度，识别和量化由于股票价格波动所蕴含的系统性风险溢价。

### 2.1 BETA - 贝塔系数

**含义**

衡量并解释"市场收益"的量化指标，对应系统性风险（不可分散风险），衡量了单只股票受市场整体波动影响的程度。

**计算逻辑**

股票收益率 r(t) 对沪深300收益率 R(t) 进行时间序列回归，取回归系数。回归时间窗口为 **504个交易日**，半衰期为 **252个交易日**。

```
r(t) = alpha + beta · R(t) + epsilon(t)
```

半衰期计算：以时间维度作为权重，当前时间权重为1，离当前时间越远权重越低。
```
λ^H = 0.5
λ = 0.5^(1/H)
```

示例：
- BETA = 1.0：当沪深300收益率上涨1%，股票收益率涨1%
- BETA = 1.5：当沪深300收益率上涨1%，股票收益率涨1.5%

**注**：BETA 可以为负值，表示股票与市场的方向关系。比如黄金，当经济恐慌时，价格通常上涨。

**所用字段**

- 股票收盘价涨跌幅
- 指数（沪深300）收盘价涨跌幅

---

### 2.2 HSIGMA - 历史残差波动率

**含义**

Hist Sigma 用于衡量股票的特异性风险（公司特有），即股票价格波动中无法被系统性风险（Beta因子）解释的那部分特异性波动。

**计算逻辑**

在计算 BETA 所进行的时间序列回归中，取回归残差收益率的波动率：

```
HSIGMA = std(epsilon)
```

**所用字段**

- 股票收盘价涨跌幅
- 指数（沪深300）收盘价涨跌幅

---

### 2.3 DASTD - 日收益率波动率

**含义**

Daily Std 衡量的是股票的总风险，反映股票价格最直观的'总波动'，这个波动既包含了随大盘起伏的系统性风险部分，也包含了公司自身特质性事件带来的波动部分。

**计算逻辑**

日收益率在过去 **252个交易日** 的波动率，半衰期 **42个交易日**：

```
DASTD_i = √252 × √[Σ(w_t · (r_{i,t} - r̄_{i,w})^2) / Σ(w_t)]
```

其中：
- `w_t = 0.5^(t/H)`：半衰期权重
- `r_{i,t}`：股票 i 在 t 日的收益率
- `r̄_{i,w}`：加权平均收益率

**所用字段**

- 股票收盘价涨跌幅

---

### 2.4 CMRA - 累积收益范围

**含义**

Cumulative Range 用于衡量股票在较长一段时间内价格波动的极端幅度，捕捉投资者在持有股票期间可能经历的"最大心理落差"（最大回撤/最大上涨）。

**计算逻辑**

Z(T) 为过去 T 个月累积对数收益率（每个月包含21个交易日）：

```
Z(T) = Σ_{t=1}^{T} [ln(1 + r_t)]
```

其中 `r_t` 为股票在 t 月的收益率。

累积收益范围：
```
CMRA = Z_max - Z_min
```

其中：
- `Z_max = max{Z(T)}`
- `Z_min = min{Z(T)}`
- `T = 1, ..., 12`

**注**：累计收益率的计算是 (1+r1)(1+r2)...(1+rT) 连乘，取对数之后为 sum[ln(1+rt)] 形式。

**所用字段**

- 股票收盘价涨跌幅

## 三、Liquidity（流动性）

流动性因子的核心作用是量化将一只股票快速变现为现金的难易程度和潜在成本，并将这种交易摩擦作为一项系统性风险因子。

### 3.1 STOM - 月换手率

**含义**

Month Share Turnover 是最近 **21个交易日**（约1个月）的日换手率之和的对数。刻画短期、高频流动性，对近期交易活跃度最敏感。

**计算逻辑**

```
STOM = ln[Σ_{t=1}^{21} (V_t / S_t)]
```

其中：
- `V(t)`：股票在 t 日的成交额
- `S(t)`：股票在 t 日的流通市值

**所用字段**

- 股票成交额
- 股票流通市值

---

### 3.2 STOQ - 季换手率

**含义**

Quarterly Share Turnover 基于过去 **3个月**（每月的STOM）计算的平均季度换手率的对数。刻画中期、平滑后的流动性，过滤了月度噪音，趋势更稳定。

**计算逻辑**

```
STOQ = ln[(1/T) · Σ_{t=1}^{T} exp(STOM_t)]
```

其中 `T = 3` 个月

**所用字段**

- 股票成交额
- 股票流通市值

---

### 3.3 STOA - 年换手率

**含义**

Annual Share Turnover 基于过去 **12个月** 计算的平均年度换手率的对数。刻画长期、结构性流动性，反映股票的固有流动性"体质"。

**计算逻辑**

```
STOA = ln[(1/T) · Σ_{t=1}^{T} exp(STOM_t)]
```

其中 `T = 12` 个月

**所用字段**

- 股票成交额
- 股票流通市值

---

### 3.4 ATVR - 年化交易量比率

**含义**

Annualized Trade Value Ratio 对过去 **252个交易日** 的日换手率进行半衰期加权求和，给予近期数据更高权重。刻画动态加权的综合流动性，结合了长期窗口和近期趋势。半衰期 **63个交易日**。

**计算逻辑**

```
ATVR = Σ_{t=1}^{252} w_t · (V_t / S_t)
```

其中：
- `V(t)`：股票在第 t 日的成交额
- `S(t)`：股票在第 t 日的流通市值
- `w(t)`：每日换手率的半衰期权重

**所用字段**

- 股票成交额
- 股票流通市值

---

## 四、Momentum（动量）

动量因子主要衡量股票在过去一段时间内的累积超额收益，捕捉价格走势的惯性。

### 4.1 STREV - 短期反转

**含义**

Short Term Reversal（短期反转）因子与 Momentum（动量）因子的逻辑恰恰相反。动量因子认为"强者恒强"，而短期反转因子则认为"涨多了必跌，跌深了必弹"。计算周期短，通常剔除最近一段时间数据。

**计算逻辑**

最近一个月的加权累积对数日收益率：

```
STREV_t = Σ_{τ=1}^{n} w_τ · [ln(1 + r_τ)]
```

其中：
- `r`：股票日收益率
- `w`：半衰指数权重
- 时间窗口 `n = 21个交易日`
- 半衰期 `= 5个交易日`

**所用字段**

- 股票收盘价涨跌幅

---

### 4.2 SEASON - 季节因子

**含义**

Seasonality 捕捉股票月度日历效应，衡量股票在某些特定月份是否存在系统性的超额收益规律（如"一月效应"、"春节效应"、"年末效应"等）。

**计算逻辑**

```
SEASON_{i,m} = (1/Y) · Σ_{y=1}^{Y} [R_{i,m,y} - R_{benchmark,m,y}]
```

其中：
- `R_{i,m,y}`：股票 i 在第 y 年第 m 个月的收益率
- `R_{benchmark,m,y}`：基准市场或行业在第 y 年的第 m 个月的收益率
- `Y`：年份数，通常取 5
- `m`：月份数，取 1-12

**所用字段**

- 股票收盘价涨跌幅、指数（沪深300）收盘价涨跌幅

---

### 4.3 INDMOM - 行业动量

**含义**

Industry Momentum 旨在捕捉和量化"行业轮动"或"行业趋势延续"效应，也就是股票所属行业的上涨或下跌趋势。

**计算逻辑**

1. 个股 S 相对强度定义为：
   ```
   RS_S(t) = Σ_{τ=1}^{n} w_τ · ln(1 + r_S(τ))
   ```
   其中 `r(s)` 为日股票收益率，`w` 为半衰指数权重，时间窗口 `n = 6个月`，半衰期 `1个月`。

2. 行业 I(t) 的相对强度（流通市值对行业内个股相对强度的加权平均）：
   ```
   RS_I(t) = Σ_{i∈I(t)} c_i(t) · RS_i(t)
   ```
   其中：
   - `c_i(t)`：行业 i 内个股流通市值平方根归一化的值
   - `RS_i(t)`：股票 i 的相对强度

3. 股票 S 的行业动量（行业相对强度排除当前股票相对强度）：
   ```
   INDMOM_S(t) = RS_I(t) - c_S(t) · RS_S(t)
   ```
   其中 `c_S(t)` 是股票 S 流通市值平方根在该行业内的归一化值。

**所用字段**

- 股票收盘价涨跌幅
- 股票流通市值
- 一级行业分类

---

### 4.4 RSTR - 相对强度

**含义**

Relative Strength 衡量的是股票在过去一年（通常是 **252个交易日**）内滞后平滑的加权平均超额收益，用于捕捉股票中长期的价格惯性。

**计算逻辑**

计算非滞后的相对强度（时间窗口252个交易日，半衰期126个交易日），然后以11个交易日为窗口，计算非滞后相对强度的等权平均值进行平滑:

```
RSTR_i = Σ_{τ=t-252}^{t-22} w_τ · (r_{i,τ} - r_{m,τ})
```

其中：
- `w(t)`：衰减权重
- `r_m(t)`：指数对数收益率 `ln(p_t/p_{t-1})`
- `r_i(t)`：股票对数收益率 `ln(p_t/p_{t-1})`

**所用字段**

- 股票收盘价涨跌幅
- 指数（沪深300）收盘价涨跌幅

---

### 4.5 HALPHA - 历史阿尔法

**含义**

Historical Alpha 在计算 BETA 的回归中，取回归截距项。捕捉的是在现有风险模型（CNE6）框架下，股票历史收益率中持续存在且无法被当前因子体系解释的部分。它可能是真正的选股技能，也可能是模型未捕捉的系统性风险暴露，或是市场持续的错误定价。

**计算逻辑**

在 BETA 的回归方程 `r(t) = alpha + beta · R(t) + epsilon(t)` 中取 `alpha` 值。

**所用字段**

- 股票收盘价涨跌幅
- 指数（沪深300）收盘价涨跌幅

## 五、Quality（质量）

质量因子的核心作用是量化一家公司基本面的稳健性、盈利能力和财务健康状况。

### 5.1 Leverage（杠杆）

衡量公司的财务结构和债务压力。高杠杆公司在经济扩张期可能获得更高收益，但在市场波动或信贷收紧时股价往往面临更大的下行风险。

#### 5.1.1 MLEV - 市场杠杆

**含义**

Market Leverage 从"市场价值"的角度来衡量一家公司的负债压力，反映了资本市场对公司偿债能力的实时评估。描述在企业总价值中，优先股和长期负债所占的相对比例。

**计算逻辑**

```
MLEV = (ME + PE + LD) / ME = 1 + (PE + LD) / ME
```

其中：
- `ME` (Market Equity)：上一交易日的总市值（股东投入的市场价值）
- `PE` (Preferred Equity)：上一财政年度优先股账面价值（优先股股东投入）
- `LD` (Long-term Debt)：上一财政年度的长期负债账面价值

**注**：市值用最新数据，是为了捕捉市场对公司价值的实时评估；PE 和 LD 用上一财政年度数据。为了保证所有公司在同一时间点使用可比的数据口径，可以在每年1月1日-4月30日期间使用上上财年数据，4月30日之后使用上一财年数据。

**所用字段**

- 股票总市值
- 其他权益工具（优先股）
- 非流动负债合计

---

#### 5.1.2 BLEV - 账面杠杆

**含义**

Book Leverage 完全基于公司财务报表上的会计数值，反映了公司资产结构中债务的物理占比，而不受股市波动的直接干扰。描述的是账面价值中有多少钱是借的。

**计算逻辑**

```
BLEV = (BE + PE + LD) / BE = 1 + (PE + LD) / BE
```

其中：
- `BE` (Book Equity)：上一财政年度的普通股账面价值
  ```
  BE = 股东权益合计(不含少数股东权益) - 其他权益工具(优先股)
  ```
- `PE` (Preferred Equity)：上一财政年度优先股账面价值
- `LD` (Long-term Debt)：上一财政年度的长期负债账面价值

**所用字段**

- 股东权益合计（不含少数股东权益）
- 其他权益工具（优先股）
- 非流动负债合计

---

#### 5.1.3 DTOA - 资产负债率

**含义**

Debt To Asset Ratio 反映了公司的总资产中有多少比例是通过借债筹集的。

**计算逻辑**

```
DTOA = TL / TA
```

其中：
- `TL` (Total Liabilities)：上一财政年度总负债
- `TA` (Total Assets)：上一财政年度总资产

**所用字段**

- 负债合计
- 资产总计

---

### 5.2 Earnings Variability（盈利波动性）

衡量公司财务业绩的稳定程度。

#### 5.2.1 VSAL - 营业收入波动率

**含义**

Variation in Sales 反应公司营业收入的波动情况（收入端：市场份额、订单稳定性）。

**计算逻辑**

过去五个财年的年营业收入标准差除以平均年营业收入。除以平均收入是为了消除公司规模的影响，使不同规模公司的波动率可比。

```
VSAL = std(营业收入_5年) / mean(营业收入_5年)
```

**所用字段**

- 营业收入

---

#### 5.2.2 VERN - 盈利波动率

**含义**

Variation in Earnings 用来捕捉公司财务报表底层盈利的稳定程度（利润端：盈利质量、成本控制）。

**计算逻辑**

过去五个财年的年净利润标准差除以平均年净利润：

```
VERN = std(净利润_5年) / mean(净利润_5年)
```

**所用字段**

- 净利润（不含少数股东损益）

---

#### 5.2.3 VFLO - 现金流波动率

**含义**

Variation in Cash-Flows 反映公司经营活动产生现金的稳定性和可预测性（现金端：回款能力、造血功能）。

**计算逻辑**

过去五个财年的年经营性活动现金流标准差除以平均经营性活动现金流：

```
VFLO = std(经营现金流_5年) / mean(经营现金流_5年)
```

**所用字段**

- 经营活动产生的现金流量净额

---

#### 5.2.4 ETOPF_STD - 分析师预测盈市率标准差

**含义**

Standard deviation of Analyst Forecast Earnings-to-Price 衡量的是分析师对公司未来盈利（EPS）预测的分歧程度，用来衡量市场对该股票的预期一致性或不确定性。

**计算逻辑**

预测12个月 EPS 的标准差除以当前股价：

```
ETOPF_STD = std(预测EPS) / 股价
```

**所用字段**

- 分析师预测的每股收益（EPS）
- 股票收盘价

**状态**：⚠️ 暂无数据

---

### 5.3 Earnings Quality（盈利质量）

主要用于衡量公司利润的可靠性。在会计学中，利润并不等同于现金。

#### 5.3.1 ABS - 资产负债表应计项目

**含义**

Accruals Balance Sheet Version 揭示公司的利润中有多少是变成了真实的现金，有多少只是账面上的数字增长。描述的是卖出去的东西多大程度上拿到钱。

应计项目（Accruals）= 会计利润 - 经营现金流

**计算逻辑**

1. 资产负债表应计项目总额：
   ```
   ACCR_BS = NOA(t) - NOA(t-1) - DA(t)
   ```
   
   其中净经营资产：
   ```
   NOA = (TA - Cash) - (TL - TD)
   ```
   
   参数说明：
   - `NOA`（Net Operating Assets）：净经营资产 = 经营性资产 - 经营性负债
   - `TA - Cash`：剔除"闲钱"后的经营资产（厂房、设备等）
   - `TL - TD`：剔除"有息借款"后的经营负债（欠供应商货款、欠员工工资）
   - `TA` (Total Assets)：总资产
   - `Cash`：现金及现金等价物
   - `TL` (Total Liabilities)：总负债
   - `TD` (Total Interest-Bearing Debt)：总带息债务 = 短期借款 + 长期借款 + 一年内到期的非流动负债 + 应付债券
   - `DA` (Depreciation and Amortization)：折旧与摊销之和 = (固定资产折旧、油气资产折耗、生产性生物资产折旧) + 无形资产摊销 + 长期待摊费用摊销

2. 将负的 ACCR_BS 除以总资产 TA：
   ```
   ABS = -ACCR_BS / TA
   ```

**所用字段**

- 资产总计
- 货币资金
- 负债合计
- 短期借款
- 长期借款
- 一年内到期的非流动负债
- 应付债券
- 固定资产折旧、油气资产折耗、生产性生物资产折旧
- 无形资产摊销
- 长期待摊费用摊销

**举例说明**：
```
ACCR_BS = 会计利润中，没有变成现金的部分
会计利润 = 50万（账面上赚的钱）
经营现金流 = 40万（实际收到的钱）
应计项目 = 10万 = 会计利润 - 经营现金流
这10万变成了：存货增加（还没卖出去变现）、应收账款增加（客户还没给钱）
```

---

#### 5.3.2 ACF - 现金流量表应计项目

**含义**

Accruals CashFlow version 直观反应利润和现金流之间的缺口，缺口越大，公司的盈利质量就越值得怀疑。ACF 衡量的是经营性应计项目，回答账面上赚的钱和实际经营收到的钱差多少。

**计算逻辑**

1. 现金流量表应计项目总额：
   ```
   ACCR_CF = NI(t) - CFO(t) + DA(t)
   ```
   
   其中：
   - `NI` (Net Income)：净利润
   - `CFO` (Cash Flow from Operating Activities)：经营现金流量净额
   - `DA`：折旧与摊销之和

2. 将负的 ACCR_CF 除以总资产 TA：
   ```
   ACF = -ACCR_CF / TA
   ```

**所用字段**

- 净利润（不含少数股东损益）
- 经营活动产生的现金流量净额
- 固定资产折旧、油气资产折耗、生产性生物资产折旧
- 无形资产摊销
- 长期待摊费用摊销

---

### 5.4 Profitability（盈利能力）

主要衡量公司利用现有资源产生利润的效率。

#### 5.4.1 ATO - 资产周转率

**含义**

Asset Turnover 衡量运营效率，公司利用其总资产产生收入的能力，比率越高，说明资产运营效率越高。

**计算逻辑**

```
ATO = Sales / TA
```

其中：
- `Sales`：过去12个月的营业收入（TTM）
- `TA` (Total Assets)：最近报告期的总资产

**所用字段**

- 营业收入
- 资产总计

---

#### 5.4.2 GP - 资产毛收益率

**含义**

Gross Profitability 衡量核心盈利效率，公司运用每单位资产能创造多少毛利。剔除了销售、管理、研发等费用的影响，更能反映核心业务的盈利能力。

**计算逻辑**

```
GP = (Sales - COGS) / TA
```

其中：
- `Sales`：上一个财务年度的营业收入
- `COGS` (Cost of Goods Sold)：上一个财务年度的营业成本
- `TA` (Total Assets)：上一个财务年度总资产

**所用字段**

- 营业收入
- 营业成本
- 资产总计

---

#### 5.4.3 GPM - 销售毛利率

**含义**

Gross Profit Margin 衡量定价权与成本控制，每单位收入中利润的占比。高毛利率通常意味着强大的品牌、定价权或成本优势。

**计算逻辑**

```
GPM = (Sales - COGS) / Sales
```

其中：
- `Sales`：上一会计年度的营业收入
- `COGS`：上一会计年度的销货成本

**所用字段**

- 营业收入
- 营业成本

---

#### 5.4.4 ROA - 总资产收益率

**含义**

Return On Assets 衡量综合盈利能力，公司利用全部资产创造净利润的整体效率。是最经典的盈利能力指标。

**计算逻辑**

```
ROA = Earnings / TA
```

其中：
- `Earnings`：过去12个月的净利润（TTM）
- `TA` (Total Assets)：最近报告期的总资产

**所用字段**

- 净利润
- 资产总计

---

### 5.5 Investment Quality（投资质量）

主要衡量公司在资产扩张和资本支出方面的节奏与效率。市场会惩罚那些为了增长而过度投资、过度融资的公司，并奖励那些资本配置谨慎、注重内生增长的公司。

#### 5.5.1 AGRO - 总资产增长率

**含义**

Total Assets Growth Rate 衡量资产扩张程度，总资产增长过快的公司，可能依赖并购或重资产扩张的模式，暴露度高，视为低质量。

**计算逻辑**

```
AGRO = - (过去5年总资产对时间回归的斜率 / 平均总资产)
```

**所用字段**

- 资产总计

---

#### 5.5.2 IGRO - 股票发行量增长率

**含义**

Issuance Growth 衡量股权稀释。频繁增发、配股进行融资的公司，暴露度高，描述对外部股权融资的依赖度高。

**计算逻辑**

```
IGRO = - (过去5年流通股本对时间回归的斜率 / 平均流通股本)
```

**所用字段**

- 流通股本

---

#### 5.5.3 CXGRO - 资本支出增长率

**含义**

Capital Expanditure Growth 衡量资本开支。资本支出增速过高的公司，暴露度高，这警示可能存在过度投资。

**计算逻辑**

```
CXGRO = - (过去5年资本支出对时间回归的斜率 / 平均资本支出)
```

**所用字段**

- 购建固定资产、无形资产和其他长期资产支付的现金

---

## 六、Value（价值）

价值因子衡量的是股票的价格与其内在价值（通常由财务指标衡量）之间的偏离程度。

### 6.1 BTOP - 账面市值比

**含义**

Book to Price 衡量资产价值，股价相对于公司净资产（清算价值）的折溢价程度。BTOP越高，说明股价相对净资产越便宜，股票越偏向"价值型"；BTOP越低，说明股价相对净资产越贵，股票越偏向"成长型"。

**计算逻辑**

```
BTOP = 最近报告期普通股账面价值 / 总市值

普通股账面价值 = 股东权益合计(不含少数股东权益) - 其他权益工具(优先股)
```

**所用字段**

- 股票总市值
- 股东权益合计（不含少数股东权益）
- 其他权益工具（优先股）

---

### 6.2 ETOP - 市盈率倒数（EP比）

**含义**

Trailing Earnings-to-price Ratio 衡量盈利价值（历史），股价相对于近期已实现盈利的便宜程度。ETOP值高，股价相对盈利便宜，偏价值；ETOP值低，股价相对盈利偏贵，偏成长。

**计算逻辑**

```
ETOP = 最近12个月净利润（TTM） / 总市值
```

**所用字段**

- 股票总市值
- 净利润（不含少数股东损益）

---

### 6.3 CETOP - 现金盈利价比

**含义**

Cash Earnings To Price 使用经营性现金流除以市值，剥离了非现金会计项的干扰，更真实地反映公司的现金产生能力与股价的关系。CETOP越高，说明股价相对经营现金流越便宜，现金流价值越高，反应资本回本周期。

**计算逻辑**

```
CETOP = 过去12个月现金盈利（TTM） / 总市值
```

**所用字段**

- 股票总市值
- 经营活动产生的现金流量净额

**示例**：100投资一家店，如果1年现金盈利为20w，则回本周期为 20/100 = 5年。

---

### 6.4 EM - 企业价值倍数的倒数

**含义**

Enterprise Multiple 衡量企业整体价值，剔除资本结构影响，从全体投资者（股+债）视角看公司核心业务的回报率。EM 高，企业价值相对核心盈利便宜，投资回报率高；EM低，企业价值相对核心盈利贵，投资回报率低。

**计算逻辑**

```
EM = 上一财年的息税前利润(EBIT) / 企业价值(EV)

EV = 总市值 + 总带息债务 - 货币资金

总带息债务 = 短期借款 + 长期借款 + 一年内到期的非流动负债 + 应付债券
```

**所用字段**

- 息税前利润
- 短期借款
- 长期借款
- 一年内到期的非流动负债
- 应付债券
- 货币资金

---

### 6.5 EPIE - 分析师预测盈利收益率

**含义**

Analyst-Predicted Earnings-to-Price 捕获的是市场专业分析师对公司未来赚钱能力的"集体预期"与当前股价的比例，用来衡量股票在未来的估值性价比。

**计算逻辑**

```
EPIE = 预测12个月的盈利 / 市值
```

**所用字段**

- 分析师预测盈利
- 股票市值

**状态**：⚠️ 无数据

---

## 七、Long Term Reversal（长期反转）

长期反转捕捉股票价格在超长期（3-5年）维度上的均值回归力量。认为经历了多年持续下跌的公司，其负面因素已充分反映，价格可能已过度偏离内在价值，具备反转潜力。

### 7.1 LTRSTR - 长期相对强度

**含义**

Long Term Relative Strength 衡量股票在超长期（3-5年）维度上，其价格趋势的疲弱或超跌程度。描述长期趋势疲弱度。

**计算逻辑**

1. 计算非滞后的长期相对强度：对股票对数超额收益率进行加权求和，时间窗口 **1040个交易日**，半衰期 **260个交易日**

2. 滞后 **273个交易日**，在 **11个交易日** 的时间窗口内取非滞后值等权平均值，然后取相反数

**所用字段**

- 股票收盘价涨跌幅
- 指数（沪深300）收盘价涨跌幅

---

### 7.2 LTHALPHA - 长期历史Alpha

**含义**

Long Term Historical Alpha 衡量股票在超长期（3-5年）维度上，其经风险调整后的超额收益的缺失或落后程度。描述长期超额亏损度。

**计算逻辑**

1. 计算非滞后的长期历史 Alpha：取 CAPM 回归（见 BETA）的截距项，时间窗口 **1040个交易日**，半衰期 **260个交易日**

2. 滞后 **273个交易日**，在 **11个交易日** 的时间窗口内取非滞后值等权平均值，然后取相反数

**所用字段**

- 股票收盘价涨跌幅
- 指数（沪深300）收盘价涨跌幅

---

## 八、Growth（成长）

成长因子捕获的是公司业务扩张的速度和持续性。

### 8.1 EGRLF - 分析师预测长期盈利增长率

**含义**

Predicted growth 3 Year 描述市场前瞻性、共识性的增长预期。这是因子的核心，直接反映市场为未来支付的溢价。

**计算逻辑**

分析师预测的长期（3-5）年利润增长率

**所用字段**

- 分析师预测的长期盈利增长率

**状态**：⚠️ 无数据

---

### 8.2 EGRO - 每股收益增长率

**含义**

Historical Earnings Per Share Growth Rate 反应已实现的盈利增长趋势，衡量公司过去的增长执行力。

**计算逻辑**

```
EGRO = 过去5个财年每股收益(EPS)对时间回归的斜率 / 平均年EPS
```

**所用字段**

- 基本每股收益

---

### 8.3 SGRO - 每股营业收入增长率

**含义**

Historical Sales Per Share Growth Rate 描述已实现的营收增长趋势，衡量公司业务规模的扩张历史。

**计算逻辑**

```
SGRO = 过去5个财年每股营收对时间回归的斜率 / 平均每股营收

平均每股营收 = 营业收入 / 总股本
```

**所用字段**

- 总股本
- 营业收入

---

## 九、Dividend Yield（股息率）

### 9.1 DTOP - 股息率

**含义**

Dividend-to-Price ratio 反应历史实现的现金回报，公司过去一年真实付出的股息相对于股价的比率。这是最经典、最可靠的股息衡量指标。

**计算逻辑**

```
DTOP = 最近12个月的实际每股股息（TTM） / 上月末股价
```

**所用字段**

- 股票收盘价
- 每股股利

**状态**：⚠️ 无数据

---

### 9.2 DTOPF - 分析师预测股息率

**含义**

Analyst Predicted Dividend to Price Ratio 描述前瞻性的现金回报预期，是市场预期公司未来将支付的股息。这反映了对公司持续分红能力和分红政策稳定性的预期。

**计算逻辑**

```
DTOPF = 预测的未来12个月每股股息(DPS) / 当前股价
```

**所用字段**

- 股票价格
- 预测每股股息

**状态**：⚠️ 无数据
